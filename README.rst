mplkit
======

A gui toolkit for creating a small set of widgets using Matplotlib, should have
the same look and feel and API regarless of the Matplotlib backend.

Widgets
=======

* Button
* Horizontal Box
* Label
* Plot
* Slider
* Vertical Box
* Window

WORK IN PROGRESS!!!
